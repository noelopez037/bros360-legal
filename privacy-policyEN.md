Privacy Policy — Bros360

Last updated: March 16, 2026

Scope of the Application

Bros360 is an internal business-use application intended exclusively for the owner and authorized employees of the company. It is not available to the general public and is not offered as a service to third parties.

Information We Collect

Bros360 collects the following information for the operation of the application:

Account data: name, email address, and password (securely managed through Supabase Auth).
Business data: sales, purchases, inventory, customer, and supplier information entered by the user within the application.
Push notification token: device identifier used to send notifications related to pharmacy activity.
Device identifier: used only to associate the notification token with a specific device.

How We Use Information

The information collected is used exclusively to:

Provide pharmacy management features, including sales, inventory, purchases, and customer management.
Send push notifications related to relevant activity within the company.
Authenticate the user and securely maintain the user session.

Data Sharing

Bros360 does not sell or share data with third parties for commercial purposes. Data is processed by the following infrastructure providers:

Supabase (supabase.com) — database and authentication. Data is stored on servers located in the US-East-1 region.
Expo / Expo Push Notifications (expo.dev) — push notification delivery service.

Data Security

Data is protected through:

Encryption in transit (HTTPS/TLS).
Row Level Security policies in the database — each company can only access its own data.
Secure authentication using JWT tokens.

Data Retention

Data is retained while the account remains active. To request deletion of the account and all associated data, contact: noe@brospharma.com

User Rights

Authorized users have the right to:

Access the data recorded in the application.
Request correction of inaccurate data.
Request deletion of their data.

To exercise these rights, contact: noe@brospharma.com

Children’s Privacy

Bros360 is an internal business-use application. It is not intended for the general public or for children.

Changes to This Policy

Any updates to this privacy policy will be published on this same page along with the updated date.

Contact

For questions about this privacy policy, contact: noe@brospharma.com
